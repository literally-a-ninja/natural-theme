### Natural Theme

A beautiful Typora theme which crosses academic and casual styles.



### Custom Elements

| Element                          | Description                                             |
| -------------------------------- | ------------------------------------------------------- |
| `<box> ... </box>`               | A block with a black border + a bit of padding.         |
| `<inline-box> ... </inline-box>` | An inline-block with a block border + a bit of padding. |
